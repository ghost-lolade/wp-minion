Minion is an accessible blog theme with a sidebar, two menus, and color options.

= Description ==
Description: Minion is an accessible, responsive blog theme. It has two menus (including a social menu), color options, and flexible widget areas with room for all your widgets.

== Installation ==
1. Unzip `minion.zip` to the `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' menu in WordPress

== Known Limitations ==
-A one line menu is strongly recommended, instead of using long menus, use sub menus.

If you have any questions or suggestions for this theme, please visit https://wordpress.org/support/theme/minion.
